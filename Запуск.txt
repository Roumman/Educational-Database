1) Установите зависимости:  

   pip install -r requirements.txt

2) Перейдите в корневую папку проекта:
   
   Укажите свой путь.
   Пример:cd "C:\Users\Rumka\Trainer\windowMB"

3) Создайте и заполните базы данных:
   
   python db_initializer.py
   
  *проверьте в cmd вывод по заполнению, должно быть все заполнено*
  *создайте копию БД в папку, где у вас будет лежать сам Тренажер, либо можете скачать сразу все вайты в одну папку*

4) Запустите мобильный банк:

   python main.py

5) Перейдите в корневую папку проекта:
   
   Укажите свой путь.
   Пример: cd "C:\Users\Rumka\Trainer\Trainer"

6) Запустите тренажёр:

   python main_Trainer.py
